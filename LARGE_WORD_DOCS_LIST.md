## Common Reasons for Large File Sizes

1. **High-resolution images** - Compress or reduce resolution
2. **Uncompressed images** - Save as JPEG instead of PNG/BMP
3. **Embedded objects** - Remove unused embedded files
4. **Multiple versions of images** - Word keeps old versions
5. **Large tables with formatting** - Simplify table styling

## Tips to Reduce File Size

1. **Compress Pictures**: Select image → Format → Compress Pictures → Best for emailing (96 ppi)
2. **Delete Unused Data**: File → Info → Check for Issues → Inspect Document
3. **Save As with Compression**: File → Save As → Tools → Compress Pictures (checkbox)
4. **Remove Edit History**: Accept all track changes before saving
5. **Export/Import**: Copy content to fresh document
